# FocusPal - ADHD专注伙伴

专为ADHD用户设计的智能陪伴产品着陆页。

## 部署说明

这个网站可以通过 GitHub Pages 免费部署：

1. 创建一个新的 GitHub 仓库
2. 将所有文件推送到仓库
3. 在仓库设置中启用 GitHub Pages（选择 main 分支）
4. 网站将自动部署到 `https://your-username.github.io/repository-name`

## 功能

- 响应式设计，支持移动设备
- 邮箱收集表单（需要后端集成）
- 早期用户注册功能

## 技术栈

- HTML5 + CSS3 + JavaScript
- 无外部依赖，纯静态网站
